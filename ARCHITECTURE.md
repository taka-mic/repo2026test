## 5. 賃料計算モデル

```
入力: 建物写真
    │
    ▼ Claude Vision API
建物分析
  ・建物種別（マンション/アパート等）
  ・構造（RC造/木造 等）
  ・外観状態（新築同様〜老朽化）
  ・エリア分類（都心/副都心/近郊/郊外/地方）
    │
    ▼ 計算モデル
人口スコア算出
  ・人口密度スコア（0〜100）
  ・流入スコア    （0〜100）
  ・推定平均年収
  ・ウォーカビリティ
    │
    ▼ 多要素掛け合わせ
最適賃料 = 基準単価(エリア) × 建物状態係数
                            × 構造係数
                            × 人口密度係数
                            × 流入係数
                            × 建物種別係数
    │
    ▼
出力
  ・最適賃料（1K/1DK基準）
  ・下限〜上限レンジ
  ・間取り別（ワンルーム〜3LDK）
  ・エリア平均比較・プレミアム率
  ・AIアドバイス文
```
